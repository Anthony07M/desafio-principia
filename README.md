# Desafio Front-end - Princípia


## Stack:

```sh
Reactjs
TypeScript
Hooks
Styled-components
Axios
react-router-dom
react-icons
```

## Instalação

```sh
git clone git@github.com:Anthony07M/desafio-principia.git
```

```sh
cd desafio-principia
```

```sh
yarn install
```

```sh
yarn start
```

```sh
localhost:3000
```
